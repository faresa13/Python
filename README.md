## ▶️ Como executar

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/python-studies.git
```

2. Entre na pasta do projeto:

```bash
cd python-studies
```

3. Execute um arquivo Python:

```bash
python nome_do_arquivo.py
```
